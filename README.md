# RFE-RL test task
## To check my notes you can search `@description (review)`
## How to start 🚀
> copy `.env.example` file into `.env` \
> `yarn` \
> `yarn start:dev` \
> open `http://localhost:3000`
## How to build 👷‍♀️
> `yarn build` \
> `yarn serve dist`
## How to test 🧪
> `yarn test`
## How to lint 🏗
> `yarn lint` or `yarn lint:fix`
## How to report 📝
> `yarn report`
